## 一、双指针

### 识别特征
> 有序数组 + 两数之和/三数之和/接雨水/盛水容器/原地修改数组

### 核心套路
```
两个指针，一个从左往右，一个从右往左
根据条件决定移动哪个指针

特殊变种：快慢指针——快指针探路，慢指针记录结果位置
常用于：原地去重、移动零、删除元素
```

### 经典题 —— 盛最多水的容器
```
思路：左右指针初始在两端，每次移动较短的那一边
（因为面积由短边决定，移动短边才可能变大）
时间 O(n)，空间 O(1)
```

```java
public int maxArea(int[] height) {
    int l = 0, r = height.length - 1, max = 0;
    while (l < r) {
        max = Math.max(max, Math.min(height[l], height[r]) * (r - l));
        if (height[l] < height[r]) l++;
        else r--;
    }
    return max;
}
```

### 经典题 —— 三数之和
```
排序 + 固定一个数 + 双指针找另外两个
去重关键：跳过相邻相同值
时间 O(n²)
```

## 二、滑动窗口

### 识别特征
> 子串/子数组 + 最长/最短 + 满足某种条件
> 关键词："连续"、"子串"、"窗口"

### 核心模板
```
右指针扩张窗口 → 满足条件 → 左指针收缩窗口 → 更新答案

本质：右指针负责找可行解，左指针负责优化可行解
```

```java
// 通用模板
int l = 0;
for (int r = 0; r < n; r++) {
    // 1. 加入 nums[r]，更新窗口状态
    // 2. while (窗口满足/不满足条件) {
    //        更新答案
    //        移除 nums[l]，l++
    //    }
}
```

### 经典题 —— 无重复字符的最长子串
```
窗口内不能有重复 → 用 HashSet 或 HashMap 记字符最后出现位置
遇到重复 → 左指针跳到重复字符的下一个位置
时间 O(n)
```

```java
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> map = new HashMap<>();
    int l = 0, max = 0;
    for (int r = 0; r < s.length(); r++) {
        char c = s.charAt(r);
        if (map.containsKey(c)) {
            l = Math.max(l, map.get(c) + 1); // 跳到重复字符之后
        }
        map.put(c, r);
        max = Math.max(max, r - l + 1);
    }
    return max;
}
```

## 三、哈希表

### 识别特征
> 两数之和、是否存在、计数、去重、映射关系
> "找配对"、"是否存在"、"第一个不重复的"

### 核心套路
```
用空间换时间：把 O(n²) 暴力搜索降为 O(n)
关键想清楚：key 是什么？value 是什么？
```

### 经典题 —— 两数之和
```java
public int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>(); // value → index
    for (int i = 0; i < nums.length; i++) {
        int need = target - nums[i];
        if (map.containsKey(need)) {
            return new int[]{map.get(need), i};
        }
        map.put(nums[i], i);
    }
    return new int[]{-1, -1};
}
```

### 经典题 —— 字母异位词分组
```
key = 排序后的字符串，value = 原字符串列表
"eat" → "aet", "tea" → "aet", 两者同 key
```

## 四、二分查找

### 识别特征
> 有序数组 + 查找/搜索插入位置/找边界
> 数据有单调性，找满足条件的"第一个"或"最后一个"

### 核心模板
```java
// 找第一个 ≥ target 的位置（下界）
int l = 0, r = n - 1;
while (l <= r) {
    int mid = l + (r - l) / 2;
    if (nums[mid] >= target) {
        r = mid - 1;  // 往左找
    } else {
        l = mid + 1;
    }
}
return l; // l = 第一个 ≥ target 的位置
```

### 关键是决定"往哪边缩"
```
找第一个 ≥ target → 满足条件时往左缩 (r = mid - 1)
找最后一个 ≤ target → 满足条件时往右缩 (l = mid + 1)
```

### 变形 —— 旋转排序数组中的搜索
```
核心：mid 左右必有一半是有序的
在有序的那一半判断 target 在不在里面
不在就去另一半
```

## 五、链表

### 识别特征
> 反转、合并、找环、找交点、删除倒数第 N 个

### 万能技巧：哑节点 (dummy)
```java
ListNode dummy = new ListNode(0);
dummy.next = head;
// 操作...
return dummy.next; // 真正的头节点
```
作用：统一处理头节点被删的情况，减少 if-else。

### 经典题 —— 反转链表
```java
public ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next;  // 保存下一个
        curr.next = prev;           // 反转指向
        prev = curr;                // prev 前移
        curr = next;                // curr 前移
    }
    return prev;
}
```

### 经典题 —— 环形链表检测
```
快慢指针：快指针每次走2步，慢指针每次走1步
有环 → 一定会相遇
找环入口：相遇后，一个指针回到head，两个指针同步走，再次相遇就是入口
```

### 经典题 —— 合并两个有序链表
```
递归：谁小谁做头，头的 next = merge(小节点的next, 另一个链表)
迭代：dummy + 逐个比较 + 拼接剩余
```

## 六、二叉树

### 识别特征
> 遍历、深度/高度、路径和、对称/翻转、最近公共祖先

### 遍历三件套

```java
// 前序：根 → 左 → 右
void preorder(TreeNode root) {
    if (root == null) return;
    print(root.val);
    preorder(root.left);
    preorder(root.right);
}

// 中序：左 → 根 → 右 （BST 的中序 = 有序序列）
void inorder(TreeNode root) {
    if (root == null) return;
    inorder(root.left);
    print(root.val);
    inorder(root.right);
}

// 后序：左 → 右 → 根 （自底向上，常用于计算高度/路径）
void postorder(TreeNode root) {
    if (root == null) return;
    postorder(root.left);
    postorder(root.right);
    print(root.val);
}
```

### 核心模板 —— 深度（自底向上）
```java
public int maxDepth(TreeNode root) {
    if (root == null) return 0;
    int left = maxDepth(root.left);
    int right = maxDepth(root.right);
    return Math.max(left, right) + 1;
}
```

### 核心模板 —— BFS（层序遍历）
```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    if (root == null) return res;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while (!q.isEmpty()) {
        int size = q.size();  // 当前层的节点数
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = q.poll();
            level.add(node.val);
            if (node.left != null) q.offer(node.left);
            if (node.right != null) q.offer(node.right);
        }
        res.add(level);
    }
    return res;
}
```

### 经典题 —— 验证二叉搜索树
```
BST 性质：左子树所有节点 < 根 < 右子树所有节点
做法：递归时传上下界 [min, max]
或者：中序遍历结果必须严格递增
```

### 经典题 —— 最近公共祖先
```
三种情况：
1. p 和 q 分别在左右子树 → root 就是答案
2. p 和 q 都在左子树 → 去左子树找
3. p 和 q 都在右子树 → 去右子树找
```

## 七、动态规划

### 识别特征
> 最值问题 + 有递推关系：最长/最小/最大/能否到达
> 关键词："最长子序列"、"最少步骤"、"有多少种方式"
> 一个决策影响后续决策

### 五步法
```
1. 定义 dp[i] 含义（一维或二维）
2. 找递推公式（状态转移方程）
3. 初始化 base case
4. 确定遍历顺序
5. 返回 dp[n] 或 dp[m][n]
```

### 经典题 —— 爬楼梯
```
dp[i] = dp[i-1] + dp[i-2]  // 到第i阶 = 从i-1来 + 从i-2来
dp[0] = 1, dp[1] = 1
```

### 经典题 —— 打家劫舍
```
dp[i] = max(dp[i-1], dp[i-2] + nums[i])
// 第i间：不偷(继承i-1) vs 偷(i-2的钱+这间的钱)
```

### 经典题 —— 最长回文子串（中心扩散法更好理解）
```
中心扩散法比DP更直观：
遍历每个位置作为中心，向两边扩散
分奇偶（单字符中心/双字符中心）
时间 O(n²)，空间 O(1)
```

### 经典题 —— 背包问题
```
01背包: dp[j] = max(dp[j], dp[j-w] + v)  // j 逆序遍历
完全背包: dp[j] = max(dp[j], dp[j-w] + v) // j 正序遍历

问法识别：
"恰好装满" → dp[0]=0, 其他=负无穷
"最大价值" → dp 全初始化为 0
"求方案数" → dp[0]=1, dp[j] += dp[j-w]
```

## 八、回溯

### 识别特征
> 所有组合/排列/子集/路径
> "找出所有可能的..."、"有多少种方式..."
> 数据量小（n ≤ 20），因为复杂度通常指数级

### 核心模板
```java
void backtrack(路径, 选择列表) {
    if (满足结束条件) {
        res.add(new ArrayList<>(路径));
        return;
    }
    for (选择 in 选择列表) {
        做选择;
        backtrack(路径, 选择列表);
        撤销选择;  // 关键！恢复状态
    }
}
```

### 经典题 —— 全排列
```java
List<List<Integer>> res = new ArrayList<>();

void backtrack(int[] nums, List<Integer> path, boolean[] used) {
    if (path.size() == nums.length) {
        res.add(new ArrayList<>(path));
        return;
    }
    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;
        used[i] = true;
        path.add(nums[i]);
        backtrack(nums, path, used);
        path.remove(path.size() - 1);  // 撤销
        used[i] = false;               // 撤销
    }
}
```

### 去重技巧
```
组合问题去重：同一层不选相同值
→ 先排序，然后 if (i > start && nums[i] == nums[i-1]) continue;
排列问题去重：同一层不选相同值 + used[i-1]==false
→ if (i > 0 && nums[i] == nums[i-1] && !used[i-1]) continue;
```

## 九、贪心

### 识别特征
> 局部最优 → 全局最优
> 通常需要排序后贪心选择
> 关键词："最多会议"、"最小编号"、"最佳时机"

### 和 DP 的区别
```
能用贪心时一定用贪心（更快）
不能贪心时用 DP

判断标准：当前最优选择是否会影响后续选择？
不影响 → 贪心；影响 → DP
```

### 经典题 —— 跳跃游戏
```java
// 维护"当前能到达的最远位置"
public boolean canJump(int[] nums) {
    int maxReach = 0;
    for (int i = 0; i < nums.length; i++) {
        if (i > maxReach) return false;  // 当前位置不可达
        maxReach = Math.max(maxReach, i + nums[i]);
        if (maxReach >= nums.length - 1) return true;
    }
    return false;
}
```

### 经典题 —— 买卖股票的最佳时机（单次交易）
```
记录到当天为止的最低价格
当天利润 = 当天价格 - 最低价格
更新最大利润
```

## 十、单调栈

### 识别特征
> 找每个元素左边/右边第一个比它大/小的元素
> "每日温度"、"下一个更大元素"、"柱状图最大矩形"
> 凡是提"下一个更大/更小"，直接用单调栈

### 核心模板
```java
// 找每个元素右边的第一个更大值（单调递减栈）
Stack<Integer> stack = new Stack<>();
for (int i = 0; i < n; i++) {
    while (!stack.isEmpty() && nums[i] > nums[stack.peek()]) {
        int idx = stack.pop();
        result[idx] = nums[i];  // nums[i] 就是 idx 右边的第一个更大值
    }
    stack.push(i);
}
```

## 十一、前缀和

### 识别特征
> 子数组求和 + 次数/个数/是否存在
> "和为 K 的子数组"、"连续子数组的和"

### 核心
```
prefix[i] = nums[0] + ... + nums[i-1]
子数组 [l, r] 的和 = prefix[r+1] - prefix[l]

常用：前缀和 + HashMap
和两数之和思路一样：找 prefix[j] - prefix[i] = k
即找 prefix[i] = prefix[j] - k
```

|-----------|---------|
| "有序数组" + 查找/两数和 | **双指针/二分** |
