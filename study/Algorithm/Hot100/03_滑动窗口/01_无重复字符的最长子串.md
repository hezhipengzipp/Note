# 无重复字符的最长子串

> 难度：Medium
> 链接：https://leetcode.cn/problems/longest-substring-without-repeating-characters
> 专题：滑动窗口

## 题目描述

给定一个字符串，找出不含重复字符的最长子串的长度。

## 思路

一句话：滑动窗口 + 哈希表记录字符位置，遇到重复字符时左指针直接跳到重复位置的下一个。

## 图解

```
s = "abcabcbb"

left=0, right 遍历，map 记录字符最后出现的位置：

right=0: 'a', map={a:0}, 窗口 "a",          len=1
right=1: 'b', map={a:0,b:1}, 窗口 "ab",     len=2
right=2: 'c', map={a:0,b:1,c:2}, 窗口 "abc", len=3
right=3: 'a', 重复! left=max(0, 0+1)=1, map={a:3,b:1,c:2}, 窗口 "bca", len=3
right=4: 'b', 重复! left=max(1, 1+1)=2, map={a:3,b:4,c:2}, 窗口 "cab", len=3
right=5: 'c', 重复! left=max(2, 2+1)=3, map={a:3,b:4,c:5}, 窗口 "abc", len=3
right=6: 'b', 重复! left=max(3, 4+1)=5, map={a:3,b:6,c:5}, 窗口 "cb",  len=2
right=7: 'b', 重复! left=max(5, 6+1)=7, map={a:3,b:7,c:5}, 窗口 "b",   len=1

结果：3（"abc"）
```

## 代码

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        HashMap<Character, Integer> map = new HashMap<>();
        int maxLen = 0;
        int left = 0;

        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);
            if (map.containsKey(c)) {
                // left 跳到重复字符的下一个位置（不能回退）
                left = Math.max(left, map.get(c) + 1);
            }
            map.put(c, right);
            maxLen = Math.max(maxLen, right - left + 1);
        }
        return maxLen;
    }
}
```

## 复杂度

- 时间：O(n) - right 指针遍历一次
- 空间：O(min(m, n)) - m 为字符集大小，n 为字符串长度

## 易错点

1. **left 用 Math.max**：`left = Math.max(left, map.get(c) + 1)` 而不是直接赋值，防止 left 回退（如 "abba"，当第二个 'b' 重复时 left=2，之后 'a' 的旧位置是 0，不能让 left 回退到 1）
2. **先更新 left 再更新 map**：顺序不能反
3. **窗口长度是 right - left + 1**：不是 right - left
