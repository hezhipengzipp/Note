# 二叉树的最大深度

> 难度：Easy
> 链接：https://leetcode.cn/problems/maximum-depth-of-binary-tree
> 专题：二叉树

## 题目描述

给定二叉树，找出其最大深度（从根节点到最远叶节点的最长路径上的节点数）。

## 思路

一句话：递归，当前节点深度 = max(左子树深度, 右子树深度) + 1。

## 图解

```
       3
      / \
     9   20
        /  \
       15   7

递归展开：
maxDepth(3)
├── maxDepth(9)
│   ├── maxDepth(null) = 0
│   └── maxDepth(null) = 0
│   └── return 1
├── maxDepth(20)
│   ├── maxDepth(15)
│   │   └── return 1
│   └── maxDepth(7)
│   │   └── return 1
│   └── return 2
└── return max(1, 2) + 1 = 3
```

## 代码

```java
class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) return 0;
        return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
    }
}
```

## 复杂度

- 时间：O(n) - 遍历所有节点
- 空间：O(h) - 递归栈深度，h 为树高（最坏 O(n)）

## 易错点

1. **base case 是 null 返回 0**：不是 1，空节点没有深度
2. **最后 +1**：加上当前节点这一层
3. **可以用 BFS 层序遍历**：数层数也行，但递归更简洁
