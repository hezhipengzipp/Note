# 翻转二叉树

> 难度：Easy
> 链接：https://leetcode.cn/problems/invert-binary-tree
> 专题：二叉树

## 题目描述

翻转一棵二叉树（镜像翻转）。

## 思路

一句话：递归交换每个节点的左右子树。

## 图解

```
原树：          翻转后：
     4              4
    / \            / \
   2   7    ->    7   2
  / \ / \        / \ / \
 1  3 6  9      9  6 3  1

递归过程：
invert(4) -> 交换 left=2, right=7
├── invert(2) -> 交换 left=1, right=3
│   ├── invert(1) -> 叶子, 返回
│   └── invert(3) -> 叶子, 返回
└── invert(7) -> 交换 left=6, right=9
    ├── invert(6) -> 叶子, 返回
    └── invert(9) -> 叶子, 返回
```

## 代码

```java
class Solution {
    public TreeNode invertTree(TreeNode root) {
        if (root == null) return null;

        // 交换左右子树
        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;

        invertTree(root.left);
        invertTree(root.right);

        return root;
    }
}
```

## 复杂度

- 时间：O(n) - 每个节点访问一次
- 空间：O(h) - 递归栈深度

## 易错点

1. **先交换再递归**：也可以先递归再交换，但先交换更直观
2. **返回 root**：递归函数要返回当前节点
3. **null 判断**：base case 返回 null
