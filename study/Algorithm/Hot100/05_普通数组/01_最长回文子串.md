# 最长回文子串

> 难度：Medium
> 链接：https://leetcode.cn/problems/longest-palindromic-substring
> 专题：动态规划 / 中心扩展

## 题目描述

给定一个字符串，找到其中最长的回文子串。

## 思路

一句话：从每个字符（和每两个字符的间隙）向两边扩展，找最长回文。

## 图解

```
s = "babad"

中心扩展法，每个字符作为中心向外扩：

i=0, 'b':   扩展 -> "b"           len=1
i=1, 'a':   扩展 -> "bab"         len=3  ✓
i=2, 'b':   扩展 -> "aba"         len=3
i=3, 'a':   扩展 -> "a"           len=1
i=4, 'd':   扩展 -> "d"           len=1

也要检查偶数长度回文（两字符间隙为中心）：
间隙0-1: "ba"  -> 不是回文
间隙1-2: "ab"  -> 不是回文
...

结果："bab" 或 "aba"

---

s = "cbbd"

i=1, 'b' 和 i=2, 'b' 间隙为中心：
向左 b, 向右 b -> 相等，继续扩
向左 c, 向右 d -> 不等，停止
得到 "bb" len=2

结果："bb"
```

## 代码

```java
class Solution {
    public String longestPalindrome(String s) {
        String result = "";
        for (int i = 0; i < s.length(); i++) {
            // 奇数长度回文，以 i 为中心
            String s1 = expand(s, i, i);
            // 偶数长度回文，以 i 和 i+1 为中心
            String s2 = expand(s, i, i + 1);
            result = result.length() > s1.length() ? result : s1;
            result = result.length() > s2.length() ? result : s2;
        }
        return result;
    }

    private String expand(String s, int left, int right) {
        while (left >= 0 && right < s.length()
               && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        // 循环结束时 left 和 right 多走了一步
        return s.substring(left + 1, right);
    }
}
```

## 复杂度

- 时间：O(n²) - n 个中心，每次扩展最多 O(n)
- 空间：O(1) - 只用常数变量

## 易错点

1. **两种中心都要试**：奇数长度（单字符中心）和偶数长度（双字符间隙中心）
2. **substring 的边界**：循环结束时 left 多减了 1，right 多加了 1，所以是 `substring(left + 1, right)`
3. **expand 传参不同**：奇数传 `(i, i)`，偶数传 `(i, i+1)`
4. **返回的是子串不是长度**：题目要求返回具体字符串
