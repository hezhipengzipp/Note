# 两数之和

> 难度：Easy
> 链接：https://leetcode.cn/problems/two-sum
> 专题：数组/哈希

## 题目描述

给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出和为目标值的两个整数，返回它们的下标。

## 思路

一句话：用哈希表存已遍历的数，边遍历边查 `target - nums[i]` 是否存在。

## 图解

```
nums = [2, 7, 11, 15], target = 9

遍历过程：
i=0, nums[0]=2, need=7, map={}          -> map中没有7, 存入 {2:0}
i=1, nums[1]=7, need=2, map={2:0}       -> map中有2! 返回 [0,1]

结果：[0, 1]
```

## 代码

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        HashMap<Integer, Integer> map = new HashMap<>();
        for (int i = 0; i < nums.length; i++) {
            int need = target - nums[i];
            if (map.containsKey(need)) {
                return new int[]{map.get(need), i};
            }
            map.put(nums[i], i);
        }
        return new int[]{};
    }
}
```

## 复杂度

- 时间：O(n) - 一次遍历
- 空间：O(n) - 哈希表最多存 n 个元素

## 易错点

1. **先查后存**：必须先 `containsKey` 再 `put`，否则同一个元素会被用两次（如 `[3,2,4]` target=6 会错误返回 `[0,0]`）
2. **返回下标不是值**：题目要的是下标
3. **只有一个解**：题目保证有且仅有一个解
