# 反转链表

> 难度：Easy
> 链接：https://leetcode.cn/problems/reverse-linked-list
> 专题：链表

## 题目描述

给定单链表的头节点，反转链表，返回反转后的链表。

## 思路

一句话：三指针迭代，逐个翻转指针方向。

## 图解

```
原链表：1 -> 2 -> 3 -> 4 -> 5 -> null

prev=null  cur=1  next=2
翻转：null <- 1    2 -> 3 -> 4 -> 5 -> null
      prev=1  cur=2  next=3

翻转：null <- 1 <- 2    3 -> 4 -> 5 -> null
      prev=2  cur=3  next=4

...直到 cur=null, prev 指向新头节点

结果：5 -> 4 -> 3 -> 2 -> 1 -> null
```

## 代码

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode prev = null;
        ListNode cur = head;

        while (cur != null) {
            ListNode next = cur.next;
            cur.next = prev;
            prev = cur;
            cur = next;
        }
        return prev;
    }
}
```

## 复杂度

- 时间：O(n) - 遍历一次
- 空间：O(1) - 只用三个指针

## 易错点

1. **先存 next 再翻转**：`next = cur.next` 必须在 `cur.next = prev` 之前，否则断链
2. **返回 prev 不是 cur**：循环结束时 cur 为 null，prev 才是新头
3. **空链表处理**：head 为 null 时直接返回 null，代码已覆盖
