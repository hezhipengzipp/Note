# 两数相加

> 难度：Medium
> 链接：https://leetcode.cn/problems/add-two-numbers
> 专题：链表

## 题目描述

给定两个非空链表，表示两个非负整数。每个节点存储一位数字，且按逆序排列。将两个数相加，返回相同形式的链表。

## 思路

一句话：双指针同时遍历两个链表，逐位相加并处理进位。

## 图解

```
l1: 2 -> 4 -> 3    (342)
l2: 5 -> 6 -> 4    (465)

carry=0
第1轮：2 + 5 + 0 = 7, carry=0  -> 7
第2轮：4 + 6 + 0 = 10, carry=1 -> 0
第3轮：3 + 4 + 1 = 8, carry=0  -> 8

结果：7 -> 0 -> 8    (807 = 342 + 465)

--- 进位溢出情况 ---
l1: 9 -> 9 -> 9
l2: 1

carry=0
第1轮：9 + 1 + 0 = 10, carry=1 -> 0
第2轮：9 + 0 + 1 = 10, carry=1 -> 0
第3轮：9 + 0 + 1 = 10, carry=1 -> 0
循环结束，carry=1            -> 1

结果：0 -> 0 -> 0 -> 1
```

## 代码

```java
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(-1);
        ListNode p = dummy;
        int carry = 0;

        while (l1 != null || l2 != null || carry != 0) {
            int sum = carry;
            if (l1 != null) {
                sum += l1.val;
                l1 = l1.next;
            }
            if (l2 != null) {
                sum += l2.val;
                l2 = l2.next;
            }
            carry = sum / 10;
            p.next = new ListNode(sum % 10);
            p = p.next;
        }

        return dummy.next;
    }
}
```

## 复杂度

- 时间：O(max(m, n)) - 遍历较长链表
- 空间：O(max(m, n)) - 结果链表长度

## 易错点

1. **循环条件加 carry**：`carry != 0` 不能省，处理最高位进位（如 999 + 1）
2. **先加 carry 再算**：sum 初始值是 carry，不是 0
3. **新建节点**：不要复用原链表节点，直接 `new ListNode(sum % 10)`
4. **链表是逆序**：题目已经是逆序存储，直接从头遍历即可，不需要反转
