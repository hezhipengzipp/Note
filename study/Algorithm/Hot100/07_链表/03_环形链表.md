# 环形链表

> 难度：Easy
> 链接：https://leetcode.cn/problems/linked-list-cycle
> 专题：链表

## 题目描述

给定链表，判断是否有环。

## 思路

一句话：快慢指针，快的走两步慢的走一步，能相遇就有环。

## 图解

```
链表：1 -> 2 -> 3 -> 4 -> 2 (回到2，有环)

slow=1, fast=1
slow=2, fast=3
slow=3, fast=2 (fast绕回来了)
slow=4, fast=4 -> 相遇！有环

无环时：fast 会先到达 null
```

## 代码

```java
public class Solution {
    public boolean hasCycle(ListNode head) {
        ListNode slow = head;
        ListNode fast = head;

        while (fast != null && fast.next != null) {
            slow = slow.next;
            fast = fast.next.next;
            if (slow == fast) {
                return true;
            }
        }
        return false;
    }
}
```

## 复杂度

- 时间：O(n) - 慢指针最多走一圈
- 空间：O(1) - 两个指针

## 易错点

1. **fast 和 fast.next 都要判空**：`fast.next.next` 会空指针，所以循环条件是 `fast != null && fast.next != null`
2. **初始位置相同**：slow 和 fast 都从 head 出发，所以 while 里先移动再比较
3. **比较用 ==**：比较的是节点引用，不是值
