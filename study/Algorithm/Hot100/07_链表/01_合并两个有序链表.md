# 合并两个有序链表

> 难度：Easy
> 链接：https://leetcode.cn/problems/merge-two-sorted-lists
> 专题：链表

## 题目描述

将两个升序链表合并为一个新的升序链表。

## 思路

一句话：用哨兵节点简化头节点处理，双指针逐个比较挂到新链表。

## 图解

```
list1: 1 -> 2 -> 4
list2: 1 -> 3 -> 4

dummy -> null
  p     l1   l2

步骤：
p->next = l1(1), p=l1, l1=l1.next  -> dummy -> 1
p->next = l2(1), p=l2, l2=l2.next  -> dummy -> 1 -> 1
p->next = l1(2), p=l1, l1=l1.next  -> dummy -> 1 -> 1 -> 2
p->next = l2(3), p=l2, l2=l2.next  -> ... -> 2 -> 3
p->next = l1(4), p=l1, l1=l1.next  -> ... -> 3 -> 4
p->next = l2(4), p=l2, l2=null     -> ... -> 4 -> 4
l2 为 null, p->next = l1(剩余)      -> 无需操作

结果：1 -> 1 -> 2 -> 3 -> 4 -> 4
```

## 代码

```java
class Solution {
    public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(-1);
        ListNode p = dummy;

        while (l1 != null && l2 != null) {
            if (l1.val <= l2.val) {
                p.next = l1;
                l1 = l1.next;
            } else {
                p.next = l2;
                l2 = l2.next;
            }
            p = p.next;
        }
        // 挂上剩余部分
        p.next = (l1 != null) ? l1 : l2;

        return dummy.next;
    }
}
```

## 复杂度

- 时间：O(n + m) - 两个链表各遍历一次
- 空间：O(1) - 只用了常数个指针

## 易错点

1. **哨兵节点 dummy**：避免单独处理头节点，统一逻辑
2. **最后挂剩余**：循环结束后，`p.next = l1 != null ? l1 : l2` 直接接上，不需要再逐个遍历
3. **<= 不是 <**：相等时取 l1 保证稳定性（其实这里无所谓，但习惯用 <=）
